Project Overview -
This project analyzes retail sales data from a Superstore dataset using Microsoft Excel. 
The objective was to identify sales trends, profitability drivers, customer behavior, and regional performance through interactive dashboarding and business analysis.


Tools Used - 
• Microsoft Excel
• Pivot Tables
• Pivot Charts
• Slicers
• INDEX-MATCH
• Data Validation
• Conditional Formatting


KPIs - 
• Quantity Sold
• Total Sales
• Total Profit
• Profit Margin


Dashboard Features - 
• Monthly Sales Trend
• Category-wise Performance
• Region-wise Analysis
• Customer Segment Analysis
• Top Products Analysis
• Interactive Slicers


Key Insights -
• West region generated highest sales.
• Technology category delivered highest profitability.
• November recorded peak sales.
• High discounts negatively impacted profit margins.
